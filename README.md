# Yavalath (ヤバラス)

Webブラウザで遊べるボードゲーム「Yavalath (ヤバラス)」の実装です。
PC、スマートフォン、タブレットの各種デバイスに対応しています。

## 概要

Yavalathは、キャメロンブラウン氏によって開発された抽象戦略ボードゲームです。
本プロジェクトでは、このゲームをHTML/CSS/JavaScriptのみを用いてWebアプリケーションとして実装しました。

### 特徴
- **レスポンシブデザイン**: PCの大画面からスマホの縦画面まで、最適なレイアウトで表示されます。
- **3人対戦モード**: 通常の2人対戦に加え、3人での対戦も可能です（黒白灰）。
- **NPC対戦**: 3段階の強さを持つAIと対戦できます。
- **美しいUI**: 木目調の六角形ボードと、ガラス風の石のデザイン。

## 遊び方

1. **勝利条件**: 自分の色の石を「4つ」一列に並べると勝ちです。
2. **敗北条件**: 自分の色の石を「3つ」一列に並べてしまうと負けです。
3. **ルール**:
   - プレイヤーは交互に空いているマスに石を置きます。
   - 「4つ並べる」か、相手に「3つ並ばせる」ことを目指します。

## インストール実行方法

本ゲームは静的なWebページとして動作します。特別なサーバー構築は不要です。

1. このリポジトリをクローンまたはダウンロードします。
2. index.html をお使いのWebブラウザ（Chrome, Edge, Firefox, Safariなど）で開いてください。

## ファイル構成

- index.html: ゲームのメイン構造
- style.css: スタイルシート（レスポンシブ対応、盤面デザイン）
- pp.js: ゲームロジック（勝敗判定、AI、UI制御）
- ssets/: 画像素材（もしあれば）

## 技術スタック

- CSS3 (Flexbox, Grid, Clip-path, CSS Variables)
- Vanilla JavaScript (ES6+)
